🔧 Pasos para Configurar MySQL

  1. Instalar MySQL Server en tu sistema
  2. Crear base de datos llamada biblioteca
  3. Configurar usuario con permisos adecuados
  4. Copiar .env.example a .env y ajustar valores
  5. Ejecutar este script para crear tablas

📋 Comandos MySQL

-- Crear base de datos
CREATE DATABASE biblioteca;

-- Crear usuario (opcional)
CREATE USER 'biblioteca_user'@'localhost' IDENTIFIED BY 'tu_password';
GRANT ALL PRIVILEGES ON biblioteca.* TO 'biblioteca_user'@'localhost';
FLUSH PRIVILEGES;

⚙️ Configuración de .env

MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=tu_contraseña
MYSQL_DATABASE=biblioteca
MYSQL_PORT=3306
